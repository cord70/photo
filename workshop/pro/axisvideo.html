AxisVideo

Требования к системе

Windows 10

Для предыдущих версий Windows убедитесь, что установлен .NET Framework 4 (Client Profile)

Установка

- распаковать AMC.zip

- установить AMC: AXISMediaControlSDK_redist.exe

- распаковать AxisVideo_xxxx.zip например в c:\temp
путь не должен содержать русских букв и пробелов

- установить: setup.exe

- согласиться на непроверенного издателя

- согласиться на допуск программы AxisVideo к сети

Первая настройка программы

- запустиь AxisVideo

- Camera-AddCamera

- в строке камеры клик по номеру 00 (у следующих камер будет 01 02 и тд)

- ввести IP камеры

- В поле после номера 00 вписать имя камеры

- Camera-AddCamera и т.д.

- File-SaveSettings

Настройка положения окна видео

- Show. Если в настройках камеры не разрешен анонимный просмотр, то видео не будет

- Изменить размер окна и переместить

- File-SaveSettings

Настройка fps и разрешения

- Camera-fps-2

- в строке камеры в поле Res вписать, например: 640x480
камера должна поддерживать разрешение в указаном формате

- File-SaveSettings

Дальнейшие настройки

- Help-Setup-ava

- Help-AxisPassword- ввести пароль к камерам (пароль пользователя root).
Логин и пароль должны быть одинаковыми для всех камер.
Если разрешен анонимный просмотр (и PTZ), то пароль не требуется.
Если логин отличается от root, то логин нужно внести вручную в файл настройки AxisVideo.xml
Например user="admin"

- Перезапуск программы

- Если
- в настройках камеры есть Enable FTP server
- и "пароль к камерам" подходит к FTP
- и введен пароль настройщика (Help-Setup-ava)
то
- Разрешение можно выбрать из списка.
- После поля Res показан тип камеры. Клик по типу камеры запускает броузер

- File-SaveSettings

Видеопоток

- Camera-MediaType-mjpeg
Все камеры должны поддерживать одинаковый видеопоток.

- File-SaveSettings

Проблемы с драйверами видеопотоков может решить программа AxisStreamingAssistantSetup.msi

Потоки mpeg4 и h264 могут стабильнее работать при небольшом fps.

Лицензия

Без лицензии через 1 час
- отключается видео для камер 04 и далее
- для показа всех камер нужно перезапустить программу

Ввод лицензии

- Help-About- в нижнее поле вводим лицезионный ключ +s-eveuenev (use copy-past)

- перезапуск программы

- если ключ принят то поле ввода ключа неактивно

- указанный ключ действует до января 2018 года

- лицензия действует на данном компьютере для данного пользователя
